# Travel Booking Website

A responsive front-end travel website with pages for **home, destinations, login/sign-up, payment, and contact**.  
Built using **HTML, CSS, and JavaScript**.

## Features

- Landing page with fullscreen travel video background
- Destinations carousel with "See More" and "Book Now" actions
- Login / Sign Up page with modern overlay UI
- Simple payment page with QR scan section
- Contact page with address, phone, email and message form
- Responsive navbar with search and mobile menu

## Pages

- `index.html` – Home page (video background + hero section)
- `book.html` – Destinations slider (cards for Singapore, Bali, China, Europe, America, Kerala)
- `login.html` – Login and registration form with social icons
- `booknow.html` – Payment / QR scan instructions
- `contact.html` – Contact form and contact details

## Tech Stack

- HTML5
- CSS3 (custom layouts, animations, responsive design)
- JavaScript (navbar toggle, video slider, interactions)
- Font Awesome icons

## How to Run

1. Clone or download the repository.
2. Open `index.html` in your browser.
3. Navigate between pages using the navbar links.

No backend or database is required for this version.
